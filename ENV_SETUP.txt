IMPORTANT: CREATE A .env FILE IN THE ROOT DIRECTORY

Copy this content into a new file named ".env" in the root directory:

===================================
# ========== SERVER CONFIGURATION ==========
NODE_ENV=development
PORT=3000
BIND_HOST=0.0.0.0

# ========== DATABASE (MONGODB) ==========
# For Local Development:
MONGODB_URI=mongodb://127.0.0.1:27017/successSprout

# For MongoDB Atlas (Cloud):
# MONGODB_URI=mongodb+srv://username:password@cluster.mongodb.net/successSprout?retryWrites=true&w=majority

# ========== JWT AUTHENTICATION ==========
JWT_SECRET=your-super-secret-jwt-key-change-this-in-production

# ========== PAYPAL INTEGRATION (Optional) ==========
# Get credentials from https://developer.paypal.com/
PAYPAL_CLIENT_ID=your-paypal-client-id
PAYPAL_CLIENT_SECRET=your-paypal-client-secret
PAYPAL_MODE=sandbox

# ========== CORS CONFIGURATION ==========
CORS_ORIGIN=http://localhost:3000
===================================

YOU MUST:
1. Install MongoDB locally: https://docs.mongodb.com/manual/installation/
   OR create a free MongoDB Atlas account: https://www.mongodb.com/cloud/atlas
2. Update MONGODB_URI with your connection string
3. Generate a strong JWT_SECRET key
4. (Optional) Add PayPal credentials for payment functionality

Without a .env file, the server will not connect to the database!

QUICK START:
1. Copy the content above to a new file named ".env" in the project root
2. Update MONGODB_URI with your MongoDB connection string
3. Generate a random JWT_SECRET: use any strong random string
4. Run: npm install
5. Run: npm run dev
